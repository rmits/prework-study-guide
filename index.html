HTML

  • Hypertext Markup Languauge (HTML) gives the site or app structure.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • Elements that add content are: Text, Image, Header, and List.
  • Elements used to divide content on the page and describe content to browsers are: Meta-related, Content Division, and Semantic.
  • Each elements are usually composed of an opening and closing tag using angle brackets.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Inline CSS is used when a developer wants to style an element directly in the HTML.
  • Internal CSS Style Sheet contains CSS rule for the site in the head section of the file where rules only apply to that one page.
  • External CSS Style Sheets are mostly used, where developers keep their specific CSS rules in an external file and link it to the HTML.

Git

  • Git is a version control system that lets you manage parts of each project.
  • To access Git, we use the command line.
  • Git commands always start with git in the command line, which saves a tracks versions of your code.
  • Git only saves to the local computer, for groups to work on the same project you will need GitHub, which keeps projects in Repos stored remotely.
  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which the computer executes code in a script.
  • Java is a high-level dynamic programming language that allows users to interact with sites.
  • Using Java, one can power site with effects, interactivity, data communication, etc.
  • If HTML is the foundation of a house, and CSS is the design, then Java is everything else such as opening the door, the running water, and the lights to the home.
  • A group of values in a variable are called Arrays.
  • Array example: var shapes=["triangle", "square", "circle"] the shapes in the brackets are an Array.
  • Java is major fun!
